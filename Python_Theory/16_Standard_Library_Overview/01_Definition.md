# Python Standard Library Overview

## Definition

The **Python Standard Library** is a collection of modules and packages that come pre-installed with Python. It provides tools for common programming tasks without requiring external dependencies.

> "Batteries Included" - Python Philosophy

## Key Categories

### 1. Built-in Functions
```python
# Always available, no import needed
len([1, 2, 3])
print("Hello")
type(42)
```

### 2. Text Processing
- `string`: String constants and templates
- `re`: Regular expressions
- `textwrap`: Text wrapping and filling

### 3. Data Types
- `datetime`: Date and time handling
- `collections`: Specialized container datatypes
- `array`: Efficient arrays of numeric values

### 4. Numeric and Mathematical
- `math`: Mathematical functions
- `random`: Random number generation
- `statistics`: Statistical functions

### 5. File and Directory Access
- `os.path`: Path manipulations
- `pathlib`: Object-oriented filesystem paths
- `glob`: Unix-style pathname pattern expansion

### 6. Data Persistence
- `pickle`: Python object serialization
- `json`: JSON encoder and decoder
- `sqlite3`: SQLite database

### 7. Internet Protocols
- `urllib`: URL handling
- `http`: HTTP modules
- `email`: Email and MIME handling

### 8. Development Tools
- `unittest`: Unit testing framework
- `pdb`: Python debugger
- `timeit`: Measure execution time
