# Abstract Algebra: Annotated Examples and Use Cases

## 1. Coloring the Cube (Burnside's Lemma)

**Problem**: How many ways to color the faces of a cube with 3 colors (R, G, B) under rotation?

**Analysis of Symmetry Group $G$ (Rotations of Cube)**: $|G| = 24$.
1.  **Identity** (1 element): Fixes 6 faces. Contribution $1 \cdot 3^6$.
2.  **Axis through opposite face centers** (3 axes $\times$ 3 rotations - identity):
    *   $90^\circ$ (6 elements): Fixes 2 faces (axis), others cycle 4. Cycles=3. Contrib $6 \cdot 3^3$.
    *   $180^\circ$ (3 elements): Fixes 2 faces, others cycle 2. Cycles=4. Contrib $3 \cdot 3^4$.
3.  **Axis through opposite edges** (6 axes $\times$ 1 rotation $180^\circ$):
    *   (6 elements): Fixes 0 faces. 3 cycles of 2. Contrib $6 \cdot 3^3$.
4.  **Axis through opposite vertices** (4 axes $\times$ 2 rotations):
    *   (8 elements): Fixes 0 faces. 2 cycles of 3. Contrib $8 \cdot 3^2$.

**Calculation**:
Total = $\frac{1}{24} (3^6 + 6 \cdot 3^3 + 3 \cdot 3^4 + 6 \cdot 3^3 + 8 \cdot 3^2)$
$= \frac{1}{24} (729 + 162 + 243 + 162 + 72) = \frac{1368}{24} = 57$.

**Answer**: 57 distinct cubes.

## 2. 15-Puzzle Solvability (Group Theory Invariant)

**Problem**: Given a scrambled 15-puzzle, is it solvable?
**Invariant**: The **Parity** of the permutation of tiles plus the parity of the taxicab distance of the empty square from the bottom-right corner must be even.
**Group Theory**:
*   Swapping a tile with empty space is a transposition (odd permutation).
*   One move changes parity of permutation AND parity of distance (row+col).
*   Therefore, (Permutation Parity + Distance Parity) is **invariant** (mod 2).

## 3. Rubik's Cube Groups

**Theory**: The states of a Rubik's Cube form a Group generated by $\{F, B, L, R, U, D\}$ (Face turns).
Size is approx $4.3 \times 10^{19}$.
**God's Number**: Maximum diameter of the Cayley Graph is 20 (every state is solvable in 20 moves). Proven computationally using group theory coset concepts.

## 4. Error Correction: Reed-Solomon Code

**Theory**: Encodes message as a polynomial over Finite Field $GF(2^m)$.
**Algebra**:
*   Message $a_0, \dots, a_{k-1}$ defines $P(x) = \sum a_i x^i$.
*   Codeword is evaluations $[P(1), P(\alpha), P(\alpha^2) \dots]$.
*   Because two degree $k-1$ polynomials intersect at most $k-1$ points, this code has large Hamming Distance.
